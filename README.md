# 🚀 Aparna Tiwari – Developer Portfolio

A modern, fully responsive personal portfolio built to showcase my projects, technical skills, and experience as a Full-Stack Developer.

This portfolio combines clean UI design, smooth animations, and interactive 3D elements to create an engaging user experience.

---

## 🌟 About The Project

This portfolio represents my journey as a Software-focused Electronics & Communication Engineering student with strong proficiency in Full-Stack Development (MERN Stack), frontend engineering, and backend systems.

It highlights:
- Real-world projects
- Technical skillset
- Professional experience
- Interactive contact functionality

---

## ✨ Key Features

- ⚡ Fully Responsive Design (Mobile + Desktop)
- 🎨 Modern UI with Tailwind CSS
- 🎬 Smooth Animations using Framer Motion
- 🌌 Interactive 3D Elements using Three.js
- 📩 Functional Contact Form powered by EmailJS
- 🧠 Clean Component-Based Architecture

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Framer Motion

**3D & Visual Effects**
- Three.js

**Backend Service**
- EmailJS (for contact form integration)
